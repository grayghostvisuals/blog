## Installation & Development

**Requires** : Node, Bundler, Grunt, Bower, Ruby, Sass, Compass, WordPress and jQuery.

**Tasks (assumes Node and Ruby are installed)**

1. Install Gemsets

    ```javascript
    bundle install
    ```

2. Install Packages

    ```javascript
    npm install && bower install
    ```

3. Compile Stylesheets and LiveReload

    ```javascript
    grunt
    ```
